Installation

To prepare project for use, perform the following steps:

  1. Ensure Carthage is installed
  2. Run ./bootstrap.sh from the Source folder.

Customization to your Project

To change from the default Trailhead naming to your project called <NewName>, perform the following steps in Xcode:

  1. Open Trailhead.xcodeproj
  2. In Project Navigator, select the Trailhead target
  3. Click Trailhead traget again, rename to <NewName>
  4. May have to press return/enter to start rename process.
  5. Let Xcode rename all items
  6. Update Signing Team if necessary
  7. Change Bundle Identifier
  8. Use Find Navigator to replace containing Trailhead with <NewName>
  9. In Project Navigator, rename folder from Trailhead to <NewName>
  10. In Project Navigator, select <NewName>, select General tab, Click Choose Info.plist File..., point it to Info.plist in <NewName> folder.
  11. Click Active Scheme selector, select Manage Schemes
  12. Rename scheme from Trailhead to <NewName>
  13. Update README.md
  14. Open Main.storyboard - it will automatically update custom class of HomeViewController to point to correct Module.
  15. …
  16. Profit!

We’re looking to automate this in the near future.
